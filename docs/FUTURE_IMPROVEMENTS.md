# Future Improvements

## Reliability

- Use different pump-ON and pump-OFF thresholds to introduce hysteresis.
- Add a flow sensor to confirm that water is actually moving.
- Add pump overcurrent and dry-run protection.
- Add a second level sensor for redundancy.
- Add a watchdog and explicit communication-fault state.
- Add a low-battery warning and controlled shutdown.

## IoT

- Add notification latching or cooldown.
- Store credentials outside the tracked source code.
- Add offline data buffering.
- Add SMS or cellular fallback.
- Add multiple-tank support.

## Power and Deployment

- Add a protected battery-backup system.
- Add solar charging through a suitable charge controller.
- Use a weather-resistant electronics enclosure.
- Separate wet-area wiring from logic electronics.

## User Interface

- Add a clear manual AUTO/ON/OFF mode rather than force-ON only.
- Add pump runtime and estimated volume.
- Add historical fault logging.
- Add local buttons for manual control.
